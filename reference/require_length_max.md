# Require Maximum Length

Validates that the value has at most length `n`.

## Usage

``` r
require_length_max(restriction, n)
```

## Arguments

- restriction:

  a `restriction` object.

- n:

  integer(1) maximum length.

## Value

The modified `restriction` object.
